Answers to the questions asked

What are sprites and why would use them?
Answer: The sprites are collection of logically separate images into a single big image (mostly vertical).
The advantages are
1) Reduced HTTP requests
2) You can compress the single image and hence total downloaded bytes would be reduced compared to multiple images
3) Overall it improves page load time
4) Easy maintenance, you don't need to maintain individual images.

How do you go about creating them?
Answer: There are tools available to do that
1) Professional graphic designers can create it using Photoshop
2) If we already have HTML page with #No of background images, there are tools like SpriteMe which can create sprite image for you. It can also create necessary CSS with classes which describes background positions for individual images
3) We can use CSS frameworks like Compass which has inbuilt features of Spriting. They have commands by which we can generate sprint images, sprite image based classes and inject in CSS

What are possible alternatives to sprites?
Answer: I know only one alternative for now, which is using data URIs in CSS.
